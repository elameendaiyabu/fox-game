# fox-game

## Tech Stack

- HTML
- CSS
- Javascript

## Prerequisites to run project

- npm

## Installation Guide

clone the repo

```sh
git clone https://github.com/elameendaiyabu/fox-game.git
```

cd into directory

```sh
cd fox-game
```

install npm dependencies

```sh
npm install
```

start development server

```sh
npm run dve
```
